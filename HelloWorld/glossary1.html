Chapter 1 Glossary & Definitions
Term Acronym/Initialism Definition
Internet An interconnected network of computer networks
HyperText Transfer Protocol HTTP A set of rules for exchanging files such as text, graphic, images, sound, video, and other multimedia files on the Web.
HyperText Markup Language HTML A set of markup symbols or codes placed in a file that is intended for display on a web page.
World Wide Web WWW The graphical user interface providing access to information stored on web servers.
World Wide Web Consortium W3C An organization taking a proactive role in developing recommendations and prototype technologies related to the web.
Americans with Disabilities Act ADA A federal civil rights law that prohibits discrimination against people with disabilities.
World Intellectual Property Organization WIPO An organization dedicated to protecting intellectual property rights internationally.
Client The computer workstations used by individuals, such as a PC on a desk.
Server Receives requests from client computers for resources, such as a file.
Client/Server Computers joined by a network.
Multi-Purpose Internet Mail Extensions MIME Rules that allow multimedia documents to be exchanged among many different computer systems.
Simple Mail Transfer Protocol SMTP The protocol used when you send e-mail messages to others.
Post Office Protocol POP3 The protocol used when you receive e-mail messages from others.
Internet Message Access Protocol IMAP A protocol that can also be used to receive e-mail messages.
File Transfer Protocol FTP A set of rules that allow files to be exchanged between computers on the internet. Used to move files directly from computer to computer as opposed to HTTP request using web browsers.
Transmission Control Protocol TCP Ensures the integrity of network communication.
Internet Protocol IP A set of rules that controls how data are sent between computers on the Internet.
Domain Name System DNS Divides the Internet into logical groups and understandable names by identifying the exact address and type of the organization.
Uniform Resource Locator URL A type of URI that represents the network location of a resource such as a web page, a graphic file, or an MP3 file.
Domain Name Locates an organization or other entity on the Internet.
Fully Qualified Domain Name FQDN The combination of host/subdomain, second-level domain, and top-level domain name.
Top-Level Domain TLD Identifies the rightmost part of the domain name (ie: .com).
Markup Language Consists of a set of directions that tell the browser software (and other user agent software that retrieves and renders web content) how to display and manage a web document.
Element Individual markup code that has a purpose and is enclosed in angle brackets.
Tag An alternative name for an Element.
Attribute Modifies or further describes the function of an element.
eXtensible HyperText Markup Language XHTML Uses the tags and attributes of HTML4 along with the more rigorous syntax of XML.
HyperText Markup Language 5 HTML5 The successor to HTML and replaces XHTML, incorporates features of both HTML and XHTML, adds new elements, provides new functionality such as form edits and native video, and is designed to be backwards compatible.
Document Type Definition DTD Identifies the version of HTML contained in the document.
DOCTYPE The DTD statement and is the first line of a web page document.
